1. Descripción General
Este sistema es una aplicación robusta basada en microservicios (Spring Boot)
que permite la generación dinámica de códigos QR a partir de texto o URLs,
 asegurando la persistencia de los datos en una base de datos relacional en memoria.


3. Arquitectura Tecnológica (Stack)
•	Lenguaje: Java 17+
•	Framework: Spring Boot 3.x (Ecosistema Spring)
•	Persistencia: Spring Data JPA con base de datos H2.
•	Generación de Imagen: Librería Google ZXing (Zebra Crossing).
•	Documentación de API: Swagger / OpenAPI 3.0.
